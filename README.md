# 📧 Email Spam Classifier

This project uses Machine Learning to classify emails as Spam or Not Spam.

## 🚀 Features
- Text preprocessing
- TF-IDF Vectorization
- Naive Bayes Model
- Confusion Matrix Visualization
- Accuracy Score
- Pie Chart Visualization

## 📊 Output
- Spam Detection
- Graphical Analysis

## 🛠 Tech Used
- Python
- Scikit-learn
- Pandas
- Matplotlib
